sub add : Public
Add route or routes.
Format: module, name1, url1, handler1, ..., nameN, urlN, handlerN.

Example:
  $this->router->add('',
  qw{
    user_read     /user/read/:id(d).html  App::Controller::User->read
    user_profile  /profile/:id(d).html    App::Controller::User::read
    _ /:module(\w)/:controller(\w+)/:action(\w+) _
  });

Access: Public

Parameters

moduleModule name (namespace prefix for controllers)
name1name of first route
url1url scheme of first route
handler1handler for user request which looks like url in first route ...
nameNname of Nth route
urlNurl scheme of Nth route
handlerNhandler for user request which looks like url in Nth route

Return: this
